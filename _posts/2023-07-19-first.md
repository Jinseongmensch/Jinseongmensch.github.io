# CU 편의점 음료 매출 회귀분석

## 1. 개요

- CU편의점의 2009~2013 음료 판매데이터를 불러와 일반 탄산음료의 매출은 어떤 독립변수의 영향을 많이 받는지 확인하기 위해 회귀분석을 시행했습니다.

## 2. 목차

- 데이터 전처리 - 이상값, 극단값, null 값 확인, 종속변수 정규성 확인

- 데이터 분석 - 상관분석, 모형 선택, 다중회귀분석

- 평가 - 회귀식 평가
